### Documentation of how to integrate real-time interaction with UNITY

**Stack**
- Node: to implent server
 - net: (node module) 
- C#: to scripting the communication
- Docker: to run nodejs

**run up server**

```
sudo docker-compose up
```
**stop server**
```
sudo docker-compose down
```
